# AI / Machine Learning Projects

This repository contains projects completed as part of an AI/ML coursework, covering:

- Machine Learning (classification, regression)
- Deep Learning & Neural Networks
- Computer Vision (CNN, Transfer Learning)
- Model evaluation and deployment concepts

## Tools & Technologies
- Python
- NumPy, Pandas, Scikit-learn
- TensorFlow / Keras
- OpenCV
- Matplotlib, Seaborn

Each folder includes Jupyter notebooks with explanations, results, and visualizations.
